# Taxfilernow.com Portfolio - Ali Awan Law Associates

This is a responsive, single-page professional web portfolio designed for **Ali Awan Law Associates**, deployed effortlessly using clean HTML5, modern utility-first CSS via **Tailwind CSS**, and pure interactive vanilla client JavaScript.

## Features Built-In
- **Responsive Navigation Framework:** Optimized layout structures designed seamlessly for mobile, tablet, and desktop viewports.
- **Dynamic Portfolio Practice Filters:** Clickable controller interfaces allowing users to parse through Tax, Corporate, or Litigation focuses instantly.
- **Dynamic Mail Engine Gateway Modals:** A standard operational popup intake wrapper generating explicit mail pipeline links down to `taxsystemonline@gmail.com`.

## Deployment Instructions on GitHub Pages
1. Create a public repository on GitHub named exactly after your desired title or profile mapping path.
2. Commit `index.html` and this `README.md` file directly to the root (`/`) of your `main` branch.
3. Head over to **Settings** -> **Pages** tabs in your target repository sidebar dashboard configuration window panels.
4. Set the **Source dropdown selection parameter** directly to **Deploy from a branch**.
5. Choose your target branch as `main` and route directory path location directly to root `/`. Hit **Save**.
6. Within minutes, your portfolio site will go online at: `https://<your-username>.github.io/<your-repo-name>/`
